# Доместик 2.0

Программа для набора песнопений древнерусского знаменного пения 
столпового и демественного распевов.

Представляет собой дальнейшее развитие проекта Василия Гладышева 2018 г.

Написана на React 9 с применением библиотек React Redux, React Redux Form.

Список изменений и доработок:

* Переработана наработанной за последние 8 лет база элементов под шрифт «Mezenetz Unicode». Весь смысл переработки изначально заключался в переводе программы на использование шрифта Unicode, который давно является современным стандартом. По ходу работы мне показалось целесообразным добавить в базу новые поля: код ZF и мелодическую расшифровку для каждого элемента. Это позволило внести новые возможности: вывод мелодии на пятилинейный нотный стан в виде двоезнаменника, а также онлайн-проигрыватель.
* Создана база элементов демественного пения на основе азбуки, представленной в книге «Лексикология демественного пения» Галины Андреевны Пожидаевой.
* Изменена логика ввода с целью более удобного ввода демества, а для ускорения набора создана возможность набора без использования мыши с помощью клавиш.
* Сохранена старая панель ввода для тех, кто предпочитает работу со старым интерфейсом.
* Добавлена возможность внесения в песнопения целых мелодических отрывков в виде попевок, лиц и фит, набранных из ряда кокизников и азбук знаменного распева. Эта работа проведена мной ранее в рамках другого проекта для использования в просветительских и научных целях.
* В связи с появившейся возможностью ввода объёмных мелодических отрывков возникла необходимость ввести режим «бесконечного холста» с последующим делением на страницы.
* Добавлена возможность сохранения набранных песнопений в базе на сайте, с возможностью выгрузки, что позволяет создать хранилище уже набранных песнопений, во избежание набора дубликатов другими участниками.
* Переделан и значительно доработан ввод текста. Все шрифты заменены на новые шрифты Unicode.
* Добавлен новый элемент — «BREAK» ДЛЯ переноса текста.
* Панель перенесена в левую часть экрана (а также может почти полностью убираться с экрана) и включает ряд новых параметров.

# Краткая справка по работе с программой

## Ввод знамён

Ввод слогов песнопения осуществляется в двумя способами:

* С помощью классической панели ввода, работа которой полностью соответствует панели ввода в Доместик 1.0.
* С помощью модального окна, вызываемого нажатием кнопки «Insert» (Windows, Linux) или кнопки «=» (Mac).
* После вызова окна выбор знамени, его дополнительных элементов, высоты, а затем и ввод текста можно выполнять тремя различными способами:
1. Клавишами клавиатуры. Список доступных клавиш и соответствующие им знамёна выводятся в поле слева.
2. Кликом мыши по списку доступных клавиш (список клавиш представляет собой кликабельные ссылки).
3. Выбором вариантов мышью в соответствующих меню.
* При вводе с помощью первых двух вариантов перемещение по полям формы осуществляется поэтапно нажатием клавиши **Tab**, а обратное перемещение по меню выполняется нажатием клавиши **Esc**. При этом список доступных клавиш в левом поле всегда отражает текущие доступные варианты. Т.е. сразу после вызова модального окна нужно выбрать нужное знамя, затем нажать клавишу **Tab** для перехода к списку элементов знамени, затем нажать клавишу **Tab** для перехода к выбору высоты, а затем и текста. При необходимости вернуться обратно, чтобы изменить сделанный выбор, нужно нажать клавишу **Esc**
* При вводе элементов знамени необязательно фиксировать все имеющиеся у него элементы — достаточно добиться того, чтобы нужный вариант знамени оказался первым элементом в списке справа. После этого можно переходить к выбору высот, а затем и текста.
* Можно также просто нажать на любой элемент списка справа. При этом знамя будет введено с прочерком вместо текста.
* Для ввода демества необходимо включить параметр «Переключить на демество» на вкладке «Настройки».

## Режим единого холста

В данной версии программы реализована возможность набора в постраничном режиме или в режиме единого холста с последующей автоматической разбивкой на страницы при отключении режима единого холста. Поэтому изначально необходимо принять решение в каком режиме следует работать:
* Работа в режиме единого холста позволяет более удобно подгружать продолжительные мелодические отрывки из базы попевок, лиц и фит.
* Включить режим можно на странице настройки параметров на вкладке «Настройки» включением движка «Единый холст».
* При отключении параметра «Единый холст» инициируется автоматическая разбивка текста на страницы.
* К сожалению, поскольку наполнение страниц основано на свободном заполнении строк по ширине конкретных знамён без точного позиционирования, разбивка может быть лишь приблизительной. Поэтому, после разбивки может возникнуть необходимость вручную доработать заполнение страниц с помощью синих кнопок переноса знамён со страницы на страницу, соответственно расположенных в нижней и верхней части страницы.

## Ввод попевок, лиц и фит

В этой версии программы возможен ввод мелодических формул (попевок, лиц и фит) знаменного распева из общей базы, содержащей около 3000 тысяч единиц.
* Для этого на панели «Инструменты» выберите Тип мелодической формулы, источник, глас, после чего станет активным список самих мелодических формул, в котором можно выбрать соответствующую попевку, лицо или фиту.
* Важным момент: Большинство мелодических формул имеет смысл вводить только в разводе, для чего необходимо установить флажок «Вставить "в разводе"».

## Двоезнаменник

Можно настроить вывод в виде двоезнаменника, для чего нужно включить соответствующий режим на вкладке «Настройки»
При этом над знамёнами будет выведен пятилинейный нотный стан с разводом знамён на ноты.

## Проигрыватель

Песнопения можно проиграть с помощью миди-проигрывателя. По умолчанию проигрывается только одно знамя. При необходимости проиграть отрывок или всё песнопение целиком нужно включить параметр «Проигрывать абзац» на вкладке «Настройки». После этого нужно будет сначала отметить конечный слог проигрываемого отрывка с помощью правой нижней кнопки из состава всплывающих кнопок на конкретном знамени. Знамя будет отмечено цветом. Затем нужно выбрать знамя, с которого начнётся воспроизведение, и нажать на нём нижнюю левую кнопку. Отрывок будет проигрываться с подсветкой каждого знамени в процессе проигрывания.

## Загрузка в базу

Набранное песнопение можно загрузить в базу песнопений на сайте. Для этого необходимо выполнить следующие действия:
* Нажать на кнопку «Загрузка в базу»
* Откроется модальное окно, в котором можно выбрать или создать книгу. Выбрать можно только свою книгу.
* Сделать это можно только после авторизации на сайте.
* Ввести название и описание песнопения
* Загрузить песнопение на сайт.

## Выгрузка из базы

Загруженные ранее песнопения можно выгрузить в программу. Для этого необходимо выполнить следующие действия:
* Нажать на кнопку «Выгрузка из базы»
* Выбрать книгу. Возможно установить флажок «Только мои книги», чтобы открылся список только книги загруженные автором.
* Выбрать одно или несколько песнопений из книги и нажать кнопку «Загрузить»
* Песнопение будет загружено на сайт.

## Ввод текста

Введено значительно больше возможностей для ввода славянского текста. Ввод осуществляется в соответствии со стандартом HIP: http://znamen.ru/fmt-slav.htm

## Ввод перевода строки

При необходимости завершить строку без её заполнения до конца необходимо ввести элемент перевода строки, который вводится при нажатии сочетания клавиш Shift-Enter. После этого выводится символ перевода строки, который можно удалить нажатием на соответствующий всплывающий значок.
